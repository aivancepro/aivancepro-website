AIvance Pro

Politique de confidentialité

Dernière mise à jour : 7 février 2026

1. Introduction

AS Corporation (« nous », « notre ») exploite l'application mobile AIvance Pro. Cette politique de confidentialité décrit comment nous collectons, utilisons et protégeons vos informations personnelles.

2. Données collectées

Nous collectons les données suivantes :

  • Informations de compte : adresse e-mail, nom d'utilisateur, photo de profil
  • Données de fitness : programmes d'entraînement, performances, objectifs, mensurations
  • Données de communauté : messages, publications, interactions avec d'autres utilisateurs
  • Données techniques : type d'appareil, système d'exploitation, identifiants anonymes

3. Utilisation des données

Vos données sont utilisées pour :

  • Fournir et améliorer nos services de coaching IA
  • Personnaliser vos programmes d'entraînement
  • Permettre les interactions communautaires
  • Envoyer des notifications liées à votre activité
  • Améliorer les performances de l'application

4. Partage des données

Nous ne vendons jamais vos données personnelles. Vos données peuvent être partagées avec :

  • Base44 : notre fournisseur de services backend pour le stockage sécurisé des données
  • OneSignal : pour l'envoi de notifications push

5. Sécurité

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données, incluant le chiffrement des communications (HTTPS) et l'authentification sécurisée.

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données personnelles
  • Droit de rectification : corriger vos données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à aivancepro@gmail.com.

7. Conservation des données

Vos données sont conservées tant que votre compte est actif. En cas de suppression de compte, vos données seront effacées dans un délai de 30 jours.

8. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Toute modification sera publiée sur cette page avec une date de mise à jour.
